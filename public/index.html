Smart Monitoring • Realtime • Multi Platform

Monitor sistem listrik secara realtime, aman, dan mudah dipantau dari mana saja

IoT Listrik Dashboard membantu kamu memantau arus, tegangan, daya semu, histori kejadian, serta status kebocoran listrik dengan notifikasi cepat dan kontrol relay berbasis role. Project ini dibuat untuk deteksi dini kebocoran arus listrik dan monitoring kondisi kelistrikan secara terintegrasi. Sistem membaca data sensor dari perangkat IoT, mengirim data ke cloud, lalu menampilkan dashboard interaktif untuk analisis cepat dan tindakan preventif.

Fitur Lengkap

Eksplorasi seluruh kemampuan Dashboard IoT Listrik. Dirancang untuk kontrol maksimal, keamanan tinggi, dan kemudahan pemantauan.

monitoring

Realtime Monitoring

Dapatkan data arus, tegangan, daya semu, dan indikasi kebocoran secara langsung (real-time) dengan latensi sub-detik melalui Firebase RTDB.

power_settings_new

Auto Cut-off & Proteksi

Relay akan otomatis memutus aliran listrik jika mendeteksi anomali berbahaya atau kebocoran arus untuk mencegah risiko korsleting dan kebakaran.

admin_panel_settings

Role-Based Access

Akses manajemen hanya untuk admin terverifikasi (Auth Email/Pass). User biasa hanya dapat memantau data tanpa bisa melakukan kontrol relay.

history

Log & Histori Kejadian

Semua peristiwa penting seperti trigger kebocoran, perubahan status relay oleh admin, hingga error tercatat rapi secara real-time di panel History.

devices

Lintas Platform

Nikmati pengalaman yang sama di berbagai perangkat: Desktop Windows/Mac (via Electron App), Android Native (Kotlin), dan Web PWA.

robot_2

Notifikasi Multi-Channel

Alert dikirim secara real-time ke Telegram dan Discord (4 channel terpisah: alerts, relay, monitoring, logs). Dikonfigurasi langsung dari admin panel tanpa menyentuh kode.

smart_toy

Bot Telegram

Gunakan @iot_kebocoran_bot untuk menerima alert kebocoran/overload langsung ke HP Anda tanpa membuka aplikasi.

send Hubungkan Bot
forum

Discord Webhook

Notifikasi ke 4 channel Discord berbeda: #alerts #relay #monitoring #logs. Dikonfigurasi via admin settings, tanpa Firebase Billing.

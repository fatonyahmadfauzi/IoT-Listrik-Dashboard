Kalibrasi Sensor
Prosedur kalibrasi: bandingkan pembacaan ESP32 dengan alat ukur referensi.
Faktor = nilai_referensi ÷ nilai_raw_ESP32
Nilai default: 1.000 (tidak ada koreksi).
Contoh: clamp meter baca 5 A, ESP32 baca 4.8 A → faktor = 5/4.8 = 1.042
Faktor konversi ADC → Volt AC.
Contoh: voltmeter = 220 V, raw ADC RMS = 0.336 V → faktor ≈ 654.8
Konfigurasi Telegram
Aman: Token dan Chat ID Telegram disimpan di Firebase (bukan di firmware). Hanya admin yang dapat mengubahnya. ESP32 membaca nilai ini dari Firebase setiap sinkronisasi.
Dapat dari @BotFather → /newbot → copy token.
Untuk grup: awali dengan -100. Untuk pribadi: ID numerik positif.
Cara cek: kirim pesan ke bot → buka api.telegram.org/bot{TOKEN}/getUpdates

Android APK

Aplikasi native Android berbasis Kotlin dengan monitoring realtime, kontrol relay, dan notifikasi cepat.

Download APK
Min. Requirements: Android 8.0 (Oreo) atau lebih baru, 2GB RAM.

Web (PWA)

Akses dashboard dari mana saja. Bisa diinstall langsung dari browser ke layar utama (Progressive Web App) tanpa perlu App Store.

open_in_browser Buka Dashboard install_mobile Install PWA

Bekerja di Chrome, Edge, dan Safari (iPhone/iPad).

Min. Requirements: Chrome / Edge / Safari (Versi terbaru).

Aplikasi PC berbasis Electron untuk laboratorium atau monitoring skala besar terpusat.

Download Setup (x64) Download Portable

terminal

CLI Portable HACKER MODE

Dashboard versi teks baris perintah lintas platform. Berjalan ringan tanpa butuh UI instalasi (Berkas executable .exe khusus untuk Windows).

Node.js Ver (Win x64) Python Ver (Win x64)
Perintah eksekusi Windows (Buka CMD / PowerShell):
.\iot-listrik-cli-node.exe .\iot-listrik-cli-python.exe
Linux, MacOS & Termux Android: Jangan unduh executabel manual. Jalankan perintah instalasi online ini di terminal Anda:
curl -sL "https://iot-listrik-dashboard.vercel.app/downloads/cli/install.sh" | bash
Pemasang akan mengatur alias. Setelah terpasang, cukup ketik: iot-listrik
